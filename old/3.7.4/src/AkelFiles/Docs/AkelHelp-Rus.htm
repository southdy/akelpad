Замечание о совместимости

 Программа предназначена для работы в системах Windows 95/98/Me и Windows NT 4.0/2000/XP/2003.

Общие замечания

 Блокнот - замечательный редактор, без наворотов, редактирующий в простом текстовом формате и поэтому незаменимый при программировании, редактировании html-документов, да и не только. Однако тот, который входит в состав Windows, крайне неудобен в использовании в силу ряда ограничений. Данная программа претендует на то, чтобы восполнить эти недостатки.
Внимание: для того, чтобы в полной мере использовать преимущества Unicode, рекомендуется выбрать один из Unicode-шрифтов, каковыми являются большинство шрифтов TrueType (например, Courier New).

Возможности

 Поскольку программа призвана заменить блокнот и нацелена на редактирование файлов в формате plain text, она не поддерживает работу с файлами в формате RTF и т. п. - это не недостаток, а особенность. Коротко об основных возможностях:
  • Однооконный режим (SDI), многооконный режим (MDI);
  • Редактирование файлов размером более 64K (вообще, размер редактируемого файла теоретически не ограничен);
  • Полная поддержка Unicode строк на Unicode системах (NT/2000/XP/2003);
  • Работа с кодировками Unicode (UTF-16 little endian, UTF-16 big endian, UTF-8);
  • Работа с любой кодовой страницей, установленной в системе;
  • Работа с DOS/Windows и Unix форматами перевода строки (в том числе сохранение);
  • Предпросмотр открытия файлов;
  • Корректное отображение псевдографики;
  • Редактирование файлов с атрибутом "Read Only";
  • Многоуровневый откат действий;
  • Поиск/замена строк текста, escape-последовательностей, а также поиск/замена по шестнадцатеричному коду;
  • Запоминание кодировки и позиции каретки в файле;
  • Список последних открытых файлов;
  • Печать;
  • Поддержка языковых модулей;
  • Поддержка плагинов (подсветка синтаксиса, авто-дополнение, запуск скриптов, клавиатурные макросы и много больше).
Настройки сохраняются по выбору в AkelPad.ini или в реестре (HKEY_CURRENT_USER\Software\Akelsoft\AkelPad).

Параметры командной строки

 AkelPad.exe [параметры] "file1.ext" "file2.ext" ...
  • /p
    		•  - печать
  • /c+
    		•  - молча создавать новый файл, если он не существует
  • /c-
    		•  - не создавать новый файл, если он не существует
  • /b+
    		•  - молча открывать файл, если он бинарный
  • /b-
    		•  - не открывать файл, если он бинарный
  • /x
    		•  - не использовать режим совместимости с MS Notepad при обработке параметров командной строки
  • /reassoc
    		•  - переустановить ранее назначенные ассоциации типов файлов
  • /deassoc
    		•  - снять ранее назначенные ассоциации типов файлов
  • /quit
    		•  - закончить обработку параметров командной строки и выйти из программы
  • /end
    		•  - закончить обработку параметров командной строки
